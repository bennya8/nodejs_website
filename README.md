# Website Application 

一套基于NodeJS开发的企业站点的模型，坑有点大，但会坚持更新！

## 前端组件

* RequireJS
* jQuery
* Bootstrap
* Bootstrap Bootbox

## 后端组件

* Express4
* Mongo DB

## 项目特色

* CRUD DEMO
* 基于 Express 4 的 MVC 目录架构
* 中间件 用户权限过滤
* 中间件 文件上次
* Async 流程控制，避免陷入无限Callback
* 分页组件
* Grunt 自动化任务管理
